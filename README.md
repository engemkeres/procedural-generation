This project aims to help me learn procedural generation.

My [interactive site (with GitHub Pages)](https://engemkeres.github.io/procedural-generation/) and debug environment uses TypeScript, React, Vite and three.js (renders with WebGPU). The shader code is written in [TSL](https://github.com/mrdoob/three.js/wiki/Three.js-Shading-Language/), which is a node-based shader abstraction, written in JavaScript.

The `exercises` branch contains the TSL implementation of several exercises from [The Book of Shaders](https://thebookofshaders.com/), chapter 5-13.
