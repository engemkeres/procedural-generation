This project aims to help me learn procedural generation.

My [interactive site (with GitHub Pages)](https://engemkeres.github.io/procedural-generation/) and debug environment uses TypeScript, React, Vite and three.js (renders with WebGPU). The shader code is written in [TSL](https://github.com/mrdoob/three.js/wiki/Three.js-Shading-Language/), which is a node-based shader abstraction, written in JavaScript.

### Learning resources:

- [The Book of Shaders](https://thebookofshaders.com/)
- [Articles by Inigo Quilez](https://iquilezles.org/articles/)
- [Field Guide to TSL and WebGPU](https://blog.maximeheckel.com/posts/field-guide-to-tsl-and-webgpu/)
- [Three WebGPU Renderer](https://sbcode.net/threejs/webgpu-renderer/)
- [TSL wiki](https://github.com/mrdoob/three.js/wiki/Three.js-Shading-Language)
- [three.js docs](https://threejs.org/docs/)
- [TSL docs](https://www.goodtsl.com/en/docs)
